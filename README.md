# Tecnologias utilizadas

## Back end
- Java
- Spring Boot
- Maven
- H2 Database
- Postman

# Como executar o projeto

## Back end
Pré-requisitos: Java 17

```bash
# clonar repositório
git clone https://github.com/eijilucas/taskmanager

# entrar na pasta do projeto back end
cd taskmanager

# executar o projeto
./mvnw spring-boot:run
```
